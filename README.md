# 무량공처 (Unlimited Void) - 영역 전개

<img width="480" height="338" alt="image" src="https://github.com/user-attachments/assets/7102b687-ce2c-439e-bf98-16c58a586ca1" />

AI 바이브코딩으로 제작된 MediaPipe와 WebGL/Canvas를 활용하여 제작된 웹 기반의 주술회전(Jujutsu Kaisen) '무량공처' 영역 전개 페이지입니다.

## 주요 기능
- **핸드 트래킹**: 고죠 사토루의 수인(손 동작)을 인식하여 언제 어디서나 영역전개를 할 수 있습니다.

## 로컬 실행 방법
1. 저장소(Repository)를 클론(Clone)합니다.
2. 웹 브라우저(Chrome 권장)에서 `index.html` 파일을 엽니다.
3. 카메라 접근 권한을 허용합니다.
