# 무량공처 (Unlimited Void) - 영역 전개

AI 바이브코딩으로 제작된 MediaPipe와 WebGL/Canvas를 활용하여 제작된 웹 기반의 주술회전(Jujutsu Kaisen) '무량공처' 영역 전개 페이지입니다.

## 주요 기능
- **핸드 트래킹**: 고죠 사토루의 수인(손 동작)을 인식하여 영역 전개를 발동시킵니다.
- **시각 효과**: 하얗게 번지는 침식 효과, 비디오 오버레이, 파티클 워프(warp) 효과 등이 포함되어 있습니다.
- **상호작용**: 사용자의 웹캠을 활용하여 공허(Void) 속으로 몰입하는 경험을 제공합니다.

## 로컬 실행 방법
1. 저장소(Repository)를 클론(Clone)합니다.
2. 웹 브라우저(Chrome 권장)에서 `index.html` 파일을 엽니다.
3. 카메라 접근 권한을 허용합니다.
